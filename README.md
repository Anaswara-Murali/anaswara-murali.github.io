## 🚀 How to Deploy on GitHub Pages

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) and sign in (or create a free account)
2. Click the **"+"** icon → **"New repository"**
3. Name it: `portfolio`
4. Set it to **Public**
5. Click **"Create repository"**

### Step 2 — Upload the File
1. In your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop `index.html` into the upload area
3. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to your repository → **Settings** tab
2. Scroll down to **"Pages"** in the left sidebar
3. Under **"Source"**, select **"Deploy from a branch"**
4. Choose **branch: main** and **folder: / (root)**
5. Click **Save**

### Step 4 — Your site is live!
Wait 1–2 minutes, then visit:
```
https://[your-github-username].github.io/portfolio
```

## ✏️ Customizing Your Portfolio

To update content, simply edit `index.html` on GitHub:
1. Click on `index.html` in your repository
2. Click the pencil ✏️ icon to edit
3. Make changes and click **"Commit changes"**

Your site will update automatically within 1–2 minutes.
